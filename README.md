# 🍒 simple-perso

A simple and beautiful personal jekyll theme designed by [DINH Anh-Thi](http://dinhanhthi.com) based on [Bootstrap 4](https://getbootstrap.com/) & [Jekyll](https://jekyllrb.com/).

![overview](./img/defaultCoverPost.jpg)

## Features

- Social profiles.
- Learning log.
- Recent projects.
- Reading lists.
- Blog posts.
- Note taking.
- Recent photos.
- Posts search.
